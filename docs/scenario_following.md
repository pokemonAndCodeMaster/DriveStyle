# 跟车风格辨识方案细化与原理解析

## 1. 理论基础：“弹簧与跑步机”隐喻

我们将跟车过程中的加速度 $a_{actual}$ 拆解为两股力量的博弈：

$$ a_{actual}(t) \approx \underbrace{\frac{\Delta v(t)}{THW(t)}}_{\text{随动基线 (被动)}} + \underbrace{\lambda \frac{v_e(t)}{THW(t)} (THW(t) - THW_k)}_{\text{主动纠偏力 (意图)}} $$

*   **随动基线（跑步机）**：一种不带有任何主观风格的、纯物理的求生本能。前车减速（跑步机后退），自车必须给出向后的力以抵消速度差。
*   **主动纠偏力（弹簧）**：决定风格的灵魂！自车与“理想目标位置”（$THW_k$）之间连着一根弹簧。离目标远则拉拽（加速），离目标近则顶回（减速）。

## 2. 辨识核心：“多宇宙测谎仪”

在一阶段，算法不知道驾驶员内心的 $THW_k$ 到底是多少（1.0s, 1.5s 还是 2.0s）。
因此，我们设立三个**平行的虚拟宇宙**：
1. 假设驾驶员目标是 1.0s（激进派），计算理论上他应该踩出的纠偏力。
2. 假设目标是 1.5s（普通派），计算理论纠偏力。
3. 假设目标是 2.0s（保守派），计算理论纠偏力。

随后，我们将实际剥离出来的纠偏力 $F_{actual}$ 与这三个虚拟宇宙的理论纠偏力进行对比，计算滑动窗口内（如 3 秒）的**残差 (Cost)**。

**赢家通吃**：哪个假设宇宙的残差最小，说明驾驶员的真实操作最符合该宇宙的物理定律，从而赋予该时间窗口对应的风格标签。

## 3. 应对极端工况 (Badcase Handling)

在**急刹 (Emergency Braking)**等极限工况下，自车受限于车辆的物理制动极限（如 $-5.0 m/s^2$）。此时，车辆的动作完全被“生存本能”（安全约束）接管，不再体现个人的“风格偏好”。

**应对策略 (Sanity Check)**：
系统引入了意图一致性校验：`valid_mask = (F_actual * error >= -0.1)`。如果实际用力的方向与理论消除误差的方向南辕北辙（或受限于物理限幅无法纠偏），该片段会被标记为 `Invalid_Following`，有效防止了辨识算法在极限工况下的发散。
